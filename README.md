# 📊 AI Data Analyst Pro

An interactive data analysis web application built using **Streamlit**, **Pandas**, and **YData Profiling**.
This app allows users to upload datasets, visualize insights, and generate automated data reports.

---

##  Live Demo

 https://data-analyst-app-tzzed5grjxsrag6ynjsfco.streamlit.app/

---

##  Features

*  Upload CSV datasets
*  View data preview and summary statistics
*  Interactive visualizations (histograms, heatmaps)
*  Automated data profiling report
*  Fast and user-friendly interface
*  AI chatbot integration

---

##  Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Matplotlib & Seaborn
* YData Profiling

##  Installation

Clone the repository:

```bash
git clone[ https://github.com/DeepInsightCoder/DATA-ANALYST-APP
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
